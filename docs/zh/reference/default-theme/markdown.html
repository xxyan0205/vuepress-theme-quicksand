# 自定义容器

  • 使用：

    ::: <type> [title]
[content]
:::
    1
    2
    3

    type 是必需的， title 和 content 是可选的。

    支持的 type 有：

    • tip
    • warning
    • danger
    • details
    • CodeGroup 和 CodeGroupItem 的别名：
      • code-group
      • code-group-item

  • 示例 1 （默认标题）：

输入

::: tip
这是一个提示
:::

::: warning
这是一个警告
:::

::: danger
这是一个危险警告
:::

::: details
这是一个 details 标签
:::
